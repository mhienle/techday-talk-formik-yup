# The Goal * Syncing state with form values * Reducing boilerplate * Validating field values * Diplaying error messages * Handling form submission * Staying sane
# (The) One solution: `formik` and `yup` * Helpers for syncing/reading/updating state, errors, ... * `setFieldValue` * `setFieldError` * `setFieldTouched` * `resetForm` * ... * Sensible defaults * `yup` as `formik`'s _recommended_ (not required) solution for form validation
